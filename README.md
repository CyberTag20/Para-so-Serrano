# Paraíso Serrano

This is a simple website for a fictional hotel called "Paraíso Serrano".

## Features

*   **Theme Switcher:** Allows the user to toggle between a light and dark theme.
*   **Language Switcher:** Allows the user to toggle between English and Portuguese.
*   **Image Switcher:** Allows the user to change the main image of the website.

## How to use

1.  Open the `index.html` file in your browser.
2.  Click the "Mudar Tema" button to switch between the light and dark themes.
3.  Click the "Mudar Idioma" button to switch between English and Portuguese.
4.  Click the "Mudar Imagem" button to change the main image of the website.
